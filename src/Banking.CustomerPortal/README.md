# Banking.CustomerPortal

Customer-facing React + TypeScript portal for the BasicBankingSystem demo.

## Current Scope

- Demo customer sign-in using existing customer records
- Dashboard with portfolio totals and recent activity
- Account list and balance inspection
- Activity history grouped by date
- Customer profile display

## Local Run

```powershell
cd src/Banking.CustomerPortal
npm install
npm run dev
```

The Vite server proxies API requests to the local backend services on ports `5101-5104`.
